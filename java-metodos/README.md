## Java métodos

A classe é definido por atributos e métodos. Os métodos são funções ou ações e subrotinas que estão disponíveis dentro das nossas ações.

- um método deve ser nomeado como verbo
- seguir padrão camelCase
- 

- `OBS: em java não existe método global, então todos os métodos devem ser feitos dentro de uma classe.`

- `OBS: caso um método não retorne qualquer valor, ele deve ser representado pela palavra void.`

- `OBS: se o método conter a palavra throws Exception isso indica que ao utilizar o método pode gerar uma exceção.`

- `OBS: private void é quando o método não pode ser visto por outras classes`
